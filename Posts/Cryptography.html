Overview

The purpose of this tool is to use the python cryptography library to generate Fernet and RSA key pairs and encrypt/decrypt files through the command line. Here's how to use it. The program requires an input file and either an encryption method or a decrypt key. Let's try encrypting this cute cat.

The program takes an input file, cat.jpg, and since the -s flag is used, the encryption type is symmetric, and the output filename is encrypted.jpg. The encrypted file can no longer be opened and the encryption/decryption key is written to sym.key. Here's the decryption syntax.

The program takes in the encrypted file as an input, and uses the -d flag and argument to decrypt the input. The file original contents are restored as they have the same md5 sum. To encrypt and decrypt a file using RSA keys, simply use the -a flag instead of the -s flag. The private key is written and the encrypted script is written into test.py.enc. For large files, instead of directing the file through RSA, it is a better to encrypt the file through symmetrical encryption, and then encrypt the symmetrical key with the RSA encryption.
